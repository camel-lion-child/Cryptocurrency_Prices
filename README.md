Crypto Market Dashboard (Top 50 + USDT Dominance Alert)
Real-time cryptocurrency prices using CoinGecko API — Streamlit App

**Overview

This project is a real-time crypto market dashboard built with Streamlit and powered by the CoinGecko API. It tracks:

Top 50 cryptocurrencies by market cap

Live price, 24h change, market cap, volume

USDT dominance and sends an alert when dominance crosses a threshold

The goal is to create a clean, lightweight Python + API + Streamlit application suitable for Data Engineering + DeFi analytics.

**Features
✔ Top 50 Crypto Prices (Real-time)

Price (USD)

24h % change

Market Cap

Volume 24h

Rank

Charts (optional)

✔ USDT Dominance Monitor

Calculates USDT market cap relative to total market cap

Sends an alert in the app when dominance goes above a custom threshold

Useful for detecting risk-off market conditions

✔ Streamlit UI

Clean, interactive interface

Auto-refresh button

Chainable filters

**Purpose

This project demonstrates an API-driven real-time dashboard, combining:

Python

API data extraction

Real-time processing

Basic financial indicators (dominance)

Streamlit app deployment

Perfect for Data Engineering + DeFi analytics applications.
